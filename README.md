# Portfolio Website

This project is based on and includes modified code from the **Alvalens Porto 2** template  
(https://github.com/Alvalens/Alvalens-porto-2-nextJs), licensed under GPL-3.0.

Large portions of the original codebase have been significantly rewritten or replaced, including:
- Page structure, layout, and component architecture  
- Styling and responsive behavior (Tailwind)  
- Project card system and animations  
- Data loading and project organization  
- Overall design, content, and user experience  

The required GPL-3.0 license file is included to comply with the terms of the original template.

---

## Features

- Smooth animations with Framer Motion  
- Responsive sections for Home, About, Projects, and Contact  
- Dynamic project data powered by JSON  
- Fully redesigned UI and project display system  
- Custom components throughout the site
